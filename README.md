# 乐宝闪送项目说明
一、项目概述
乐宝闪送是一款面向即时配送场景的线上生活服务平台，聚焦“线上下单、极速送达”的用户需求，提供餐饮、生鲜、零食、药品等多品类商品的即时配送服务，同时覆盖跑腿代办、会员权益管理、订单与售后管理、用户中心设置等增值服务。平台以**高效配送、丰富供给、优惠活动、便捷服务、完善售后、个性化设置**为核心竞争力，打造一站式的本地生活消费入口，为用户提供便捷、多样的即时购物与代办体验。

本项目为平台的Web端前端页面，覆盖首页导航、活动专区、品类专区、用户权益、跑腿服务、商家详情（含肯德基、每日生鲜典型商家页）、订单管理、消息通知、用户中心、售后申请、评价管理、客服帮助、账户设置等全链路场景，兼顾功能完整性与视觉体验，适配用户从浏览、下单到售后、账户管理的全流程服务决策链路。

---
二、核心功能模块说明
1. 首页主界面（核心入口）
首页作为用户进入平台的第一触点，承担着导航、信息触达、场景引导三大核心作用：
- **顶部导航栏**：品牌标识+核心功能入口，支持“首页/订单/登录”快速切换，同时集成定位（默认北京市朝阳区）、搜索栏（支持商家/商品搜索）、消息通知入口，满足用户快速定位、搜索、查看订单与消息的基础需求。
- **场景分类导航**：分为“活动专区”“品类专区”“工具专区”三大模块：
  - 活动专区：满减、折扣、新店、夜宵、下午茶、生鲜超市，直接引导用户进入对应优惠场景；
  - 品类专区：美食、奶茶、生鲜、超市、药品、水果、蛋糕、快餐，覆盖用户高频消费品类；
  - 工具专区：红包、优惠券、会员、跑腿，为用户提供平台权益与增值服务入口。
- **商家与商品推荐区**：分为“附近商家、人气排行、新店推荐、超值套餐”四大板块：
  - 附近商家：展示周边热门连锁品牌（如肯德基、麦当劳、星巴克），标注评分、配送时长、标签，帮助用户快速选择；
  - 人气排行：基于平台用户消费数据，展示热门品牌（如必胜客、海底捞、喜茶），降低用户决策成本；
  - 新店推荐：推送平台新入驻商家，同步展示“新店”标识，引导用户尝试新供给；
  - 超值套餐：直接展示热门商品套餐（如香辣鸡腿堡套餐、奥尔良烤翅套餐），标注原价与优惠价，刺激用户下单。

2. 活动专区子页面
针对首页的六大活动场景，平台设计了独立的活动专题页，强化活动心智与转化：
- **满减活动页**：明确标注“全场满减，最高立减50元”，按档位区分“满50减15/满100减30/满200减50”，同步展示参与活动的商家及专属优惠标签（如“新用户立减20”）。
- **折扣活动页**：主打“低至5折起”，覆盖限时特惠、爆款直降场景，展示参与折扣的商家与对应优惠（如限时8折、新品8折）。
- **新店活动页**：聚焦新入驻商家，提供“新店特惠、首单立减20、新用户专享”权益，帮助新店快速获客，同时为用户提供尝鲜福利。
- **夜宵活动页**：面向22:00-04:00的夜间消费场景，主打“深夜配送、暖胃夜宵”，适配海底捞、烧烤类商家，同步推出满减与深夜专属优惠。
- **下午茶活动页**：针对14:00-17:00的下午茶时段，主打“买一送一、甜品特惠”，覆盖咖啡、奶茶类商家，匹配用户午后休闲消费需求。
- **生鲜超市活动页**：主打“新鲜直达、当日达”，提供生鲜、水果、蔬菜等品类服务，标注品质保障权益，同步推出满减优惠，满足用户即时采购需求。

3. 品类专区子页面
为满足用户精细化的消费需求，平台为每个高频品类设计了独立的分类页，支持细分筛选与商家展示：
- **美食分类页**：支持“全部/中餐/西餐/日料/韩餐”筛选，展示肯德基、麦当劳等品牌，标注评分、配送时长与优惠活动（如“满50减15免配送费”），满足正餐消费需求。
- **奶茶分类页**：支持“全部/经典奶茶/水果茶/芝士奶盖/特色饮品”筛选，覆盖喜茶、茶颜悦色等品牌，同步展示“第二杯半价”“买一送一”等饮品专属优惠。
- **生鲜分类页**：支持“全部/肉类/海鲜/蔬菜/蛋类”筛选，展示每日生鲜、鲜果园等商家，标注“当日达”“品质保证”等服务承诺，满足用户食材采购需求。
- **超市分类页**：支持“全部/零食/饮料/日用品/冷冻食品”筛选，覆盖永辉超市等品牌，提供“满99减15”“特价专区”等优惠，适配日常百货采购场景。
- **药品分类页**：支持“全部/感冒用药/止痛消炎/保健品/医疗器械”筛选，展示大参林等药店，标注“满50减10会员折扣”，满足用户应急购药需求。
- **水果分类页**：支持“全部/国产水果/进口水果/礼盒装/切盒”筛选，覆盖鲜果类商家，同步展示“满80减15”等优惠，适配日常水果采购场景。
- **蛋糕分类页**：支持“全部/生日蛋糕/芝士蛋糕/慕斯蛋糕/面包甜点”筛选，展示好利来等品牌，提供“生日蛋糕8折”等专属优惠，满足节日、日常甜品消费需求。
- **快餐分类页**：支持“全部/汉堡炸鸡/披萨/中式快餐/三明治”筛选，覆盖肯德基、麦当劳等品牌，标注“新用户立减20”“第二份半价”等快餐专属优惠，适配便捷用餐场景。

4. 用户权益专区
为提升用户粘性，平台设置了独立的用户权益页面，帮助用户高效管理优惠福利与会员服务：
- **红包页面**：展示用户可使用的红包，标注红包金额、使用门槛、有效期，支持“立即使用”操作，同时区分已过期红包，方便用户直观了解权益状态。
- **优惠券页面**：按“可使用/已使用/已过期”分类管理优惠券，展示满减券、商家专享券等不同类型的优惠，标注面额、使用门槛与有效期，帮助用户合理规划优惠使用。
- **会员页面**：展示用户会员等级（如黄金会员）、积分信息与升级进度，清晰呈现会员专属权益：
  - 专属折扣：全场95折优惠；
  - 优先配送：订单优先处理；
  - 生日礼包：生日当月领取专属福利；
  - 积分加速：消费享1.5倍积分；
  - 免配送费：每月3次免运费权益；
  - 专属优惠券：每月可领取会员专属券，全方位提升用户消费体验。

5. 跑腿服务专区
为满足用户多元化的即时代办需求，平台提供了完善的跑腿服务模块：
- **服务分类页**：提供四大核心跑腿服务，覆盖不同代办场景：
  - 帮我买：代用户购买商品并送货上门；
  - 帮我送：代用户送达物品至指定地点；
  - 药品代购：专为用户提供药品代购服务；
  - 文件代送：为用户提供重要文件的代送服务。
- **跑腿订单提交页**：支持用户填写取货地址、送货地址、商品/物品信息、备注等内容，提交跑腿订单，流程清晰，适配各类代办场景。

6. 商家详情与下单流程（以肯德基、每日生鲜为典型示例）
平台为用户提供了完整的商家浏览与下单链路，以肯德基、每日生鲜为典型示例，帮助用户高效完成消费决策：
- **商家信息页**：展示商家基础信息，包括名称、营业状态、评分、月售、优惠活动、地址、联系方式、配送时长、起送价/配送费、营业时间等核心信息，帮助用户快速了解商家情况。
  - 肯德基页面：展示快餐类商家信息，提供汉堡/鸡肉、小食、饮品、套餐、甜点等分类筛选，商品列表清晰展示价格、月售数据，支持增减数量与结算操作；
  - 每日生鲜页面：展示生鲜类商家信息，提供蔬菜、水果、肉类、海鲜、蛋类等分类筛选，商品列表适配生鲜品类展示，满足用户食材采购需求。
- **商品列表页**：支持按分类筛选商品，展示商品图片、名称、标签、价格、原价、月售数据，用户可直接增减商品数量，实时查看购物车金额与起送差额，点击“去结算”即可进入下单流程。
- **店铺评价页**：展示商家综合评分及口味、服务、环境、性价比等细分维度评分，同时呈现用户真实评价，帮助用户参考他人消费体验，降低决策成本。
- **商家介绍页**：展示商家品牌背景、经营理念等详细信息，增强用户对商家的信任度。

7. 消息通知中心
平台设置了独立的消息通知弹窗，集中展示用户的订单通知、优惠通知、积分通知、活动通知等信息，支持“全部已读”“关闭”操作，确保用户不遗漏关键信息，同时提升平台消息触达效率。

### 8. 订单管理页面
为方便用户管理消费订单，平台提供了完整的订单管理功能：
- **订单分类管理**：按“全部/待付款/待接单/配送中/待收货/已完成/退款售后”对订单进行分类，用户可快速筛选不同状态的订单。
- **订单详情展示**：展示订单所属商家、订单时间、订单号、商品明细、实付金额等信息，同时支持“再来一单”“评价”“申请售后/退款”操作，覆盖订单全生命周期管理需求。
- **待付款订单页**：专门展示待付款订单，用户可直接点击“去付款”完成支付，或选择“取消订单”放弃订单，流程便捷。

### 9. 用户中心模块
平台为用户提供了功能完善的个人中心，实现账户信息与服务的一站式管理：
- **登录/注册页面**：提供用户名、密码登录入口，同时支持新用户跳转注册，保障用户账户安全与平台使用权限。
- **个人信息概览**：展示用户头像、用户名、会员等级、手机号，同时呈现账户余额、积分、优惠券数量、红包数量等核心信息，方便用户快速掌握账户状态。
- **订单管理入口**：整合“全部订单/待付款/待收货/退款售后”等订单分类入口，用户可直接跳转对应订单页面。
- **服务与设置模块**：提供“我的收藏/收货地址/我的评价/客服中心/帮助中心”等服务入口，同时支持“夜间模式/消息推送/账号安全/隐私设置/通用设置”等账户设置，满足用户个性化使用需求。
- **退出登录功能**：支持用户一键退出登录，保障账户安全。

### 10. 售后与账户管理功能
平台为用户提供了完善的售后与账户管理工具，保障用户消费权益：
- **售后申请页面**：支持用户选择售后类型（仅退款/退货退款/损坏赔付/投诉商家/骑手），填写退款金额、退款原因、问题描述并上传凭证，提交售后申请，覆盖各类售后场景。
- **余额管理页面**：展示账户余额与明细（充值、订单支付、退款记录），支持用户充值、提现操作，方便用户管理账户资金。
- **优惠券管理页面**：展示用户可使用的优惠券信息，包括面额、使用门槛、适用商家、有效期，帮助用户合理规划优惠使用。
- **收藏管理页面**：展示用户收藏的商家信息，包括商家名称、评分、地址、收藏时间，支持用户删除收藏，方便用户快速访问心仪商家。
- **地址管理页面**：支持用户添加、编辑、删除收货地址，同时可设置默认地址，适配多场景收货需求。

### 11. 用户评价与服务支持模块
平台为用户提供了完善的评价与服务支持功能，提升用户消费体验：
- **我的评价页面**：展示用户已发布的订单评价，包括商家名称、订单号、评分、评价内容与评价时间，方便用户回顾消费体验，同时为其他用户提供参考。
- **客服中心页面**：提供客服电话与服务时间，同时支持智能问答功能，用户可输入问题与客服交互，快速获取订单进度、配送时间等问题的解答。
- **常见问题页面**：汇总用户高频问题（如修改密码、取消订单、联系客服、使用优惠券、配送时间等），用户可直接查看问题解答，快速解决常见疑问。

### 12. 账户设置模块
平台为用户提供了全面的账户设置功能，满足用户个性化使用与账户安全需求：
- **账号安全页面**：支持手机绑定、邮箱绑定、登录密码修改、支付密码设置、实名认证、设备管理等操作，全方位保障用户账户安全。
- **隐私设置页面**：提供个人信息收集、位置信息、消息通知、个性化推荐、数据共享等开关设置，用户可自主管理隐私权限。
- **通用设置页面**：支持推送通知、自动登录、语言设置、主题设置、自动播放视频等功能设置，适配用户个性化使用习惯。
